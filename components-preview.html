Main Menu

Sticky navigation menu with 5 items. The "Consultation" button is highlighted as a CTA. On mobile, the menu collapses into a hamburger icon. Use the font toggle (Aa) and width toggle (Column/Full) in the top right.

Menu Items: About, Services, Contacts, Navigation, Consultation

Features:

  • Sticky positioning (stays at top while scrolling)
  • Border highlight on hover for each item
  • CTA button styling for "Consultation"
  • Mobile hamburger menu
  • Font and width toggle controls
Buttons

Rectangular buttons with filled backgrounds. Primary buttons use the accent color (dark blue). Buttons change color on hover. Up to 5 buttons can be arranged in a horizontal row.

Enhanced Links

Links are styled with the accent color and have interactive hover effects. The underline thickens on hover to clearly indicate clickability.

Standard links are styled like this: Click here to learn more about our immigration services. You can also contact our office or visit the Immigration, Refugees and Citizenship Canada website.

Links with background highlight: Hover over me to see the subtle background effect.

Font Toggle

Users can switch between serif (Times New Roman) and sans-serif (Inter) fonts. On mobile devices (<640px), sans-serif is automatically used for better readability.

Current Font Sample

The quick brown fox jumps over the lazy dog. This sentence contains every letter of the alphabet and demonstrates the current font style.

Bold text and italic text are also supported.

Try it: Click the "Aa / Aa" button in the menu to toggle between fonts.

Width Toggle

Users can switch between column view (optimal reading width, ~52rem) and full-width view. Column view is better for reading; full-width is better for viewing images or complex layouts.

Try it: Click the "Column" / "Full" button in the menu to toggle the content width.

Current mode: Column

Responsive Design

The layout adapts to different screen sizes. Resize your browser window to see mobile (≤640px), tablet (641-1024px), and desktop (≥1025px) layouts.

Responsive Features

  • Menu collapses to hamburger on mobile
  • Sans-serif font auto-enabled on mobile for better readability
  • Reduced spacing and font sizes on smaller screens
  • Button rows stack vertically on mobile
  • Indentation reduced on mobile to preserve reading space
